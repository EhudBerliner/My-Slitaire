## ⚙️ תכונות מתקדמות

### 1. מנגנון Force Reinstall
**מה זה?** כפתור "פאניק" שמאפס את כל האפליקציה.

**איך להשתמש:**
1. לחץ על מספר הגרסה (למשל: v2.1.0) בפינה השמאלית העליונה
2. אשר את האיפוס
3. האפליקציה תמחק את:
   - כל ה-Cache
   - Service Workers
   - localStorage (כולל משחקים שמורים)
4. רענון אוטומטי עם קבצים חדשים מהשרת

**מתי להשתמש?**
- כשהמשחק לא טוען כראוי
- אחרי עדכון שלא עבד
- כשיש באגים מוזרים

### 2. מערכת העדכונים האוטומטית

**איך זה עובד?**
1. Service Worker בודק עדכונים כל שעה
2. כשיש גרסה חדשה - הורדה ברקע
3. עדכון מופעל ברענון הבא
4. Badge הגרסה משתנה לכתום = יש עדכון

**אסטרטגיית Cache:**
- Cache-First: קבצים נטענים מהמטמון (מהיר!)
- Network Fallback: אם אין במטמון, מהשרת
- Auto-Update: מטמון מתעדכן אוטומטית

### 3. תמיכה Offline מלאה

האפליקציה עובדת **לחלוטין ללא אינטרנט** לאחר הטעינה הראשונה:
- ✅ כל הקלפים
- ✅ כל האנימציות
- ✅ שמירה וטעינה
- ✅ סטטיסטיקות

## 🐛 פתרון תקלות

### המשחק לא נטען?
1. רענן את הדף (F5)
2. נקה Cache: Ctrl+Shift+Delete
3. השתמש ב-Force Reinstall (לחץ על הגרסה)

### הקלפים לא זזים?
1. בדוק שאתה לוחץ על קלף גלוי
2. וודא שהמהלך חוקי
3. נסה לחיצה כפולה להעברה אוטומטית

### הגרסה לא מתעדכנת?
1. סגור את כל הטאבים של המשחק
2. פתח מחדש
3. אם לא עובד - Force Reinstall

### על iOS - אין התקנה?
Safari דורש:
1. פתח ב-Safari (לא Chrome!)
2. לחץ על Share (ריבוע עם חץ)
3. גלול ל-"Add to Home Screen"

```
index.html          - UI Structure + CSS
├── game.js         - Game Logic
│   ├── SolitaireGame      - Game state & rules
```
