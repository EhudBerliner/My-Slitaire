# סוליטר פרו - Solitaire Pro PWA

## גרסה 2.0.0

משחק סוליטר קלאסי (Klondike) עם ממשק משתמש מודרני, אנימציות חלקות ותמיכה מלאה ב-PWA.

## ✨ תכונות עיקריות

### 🎮 משחקיות
- **מנוע משחק מלא**: מימוש מושלם של חוקי סוליטר Klondike
- **שתי רמות קושי**: שליפה של 1 או 3 קלפים
- **Drag & Drop חלק**: תמיכה מלאה במגע ועכבר
- **Undo ללא הגבלה**: עד 50 מהלכים אחורה
- **שמירה אוטומטית**: המשך מאותה נקודה בכל עת
- **מערכת רמזים**: קבלת הצעות למהלכים אפשריים
- **סיום אוטומטי**: השלמת המשחק כשאפשרי

### 🎨 עיצוב וגרפיקה
- **אנימציות 60 FPS**: תנועות קלפים חלקות
- **5 ערכות נושא**: ירוק קלאסי, כחול, אדום, סגול, כהה
- **4 עיצובי גב קלף**: כחול, אדום, ירוק, משובץ
- **אנימציית ניצחון**: קונפטי ופידבק ויזואלי
- **עיצוב רספונסיבי**: התאמה מושלמת לכל מסך

### 📱 PWA ומובייל
- **התקנה למסך הבית**: עבודה כאפליקציה עצמאית
- **תמיכה באופליין**: שחק בלי אינטרנט
- **ניהול גרסאות**: עדכונים אוטומטיים
- **איפוס קשיח**: לחיצה על מספר הגרסה
- **התאמה ל-Notch**: תמיכה בטלפונים מודרניים
- **משוב הפטי**: רטט קל במהלכים (אופציונלי)

### 📊 סטטיסטיקות
- משחקים ששוחקו ונוצחו
- אחוז ניצחונות
- שיא ניקוד
- זמן ממוצע למשחק

### ♿ נגישות
- **קלפים גדולים**: מצב נגישות למי שצריך
- **תמיכה בקיצורי מקלדת**:
  - `Space` - שלוף קלף
  - `Ctrl/Cmd + Z` - ביטול מהלך
  - `H` - רמז
  - `N` - משחק חדש
- **ניגודיות גבוהה**: תמיכה במצב ניגודיות גבוהה
- **ביטול אנימציות**: עבור מי שרגיש לתנועה

## 🚀 התקנה והרצה

### דרישות
- שרת HTTPS (נדרש ל-PWA)
- דפדפן מודרני התומך ב-Service Workers

### שלבים
1. העלה את כל הקבצים לשרת HTTPS
2. גש ל-URL של האפליקציה
3. האפליקציה תיטען ותתקין את ה-Service Worker
4. ניתן להתקין למסך הבית דרך תפריט הדפדפן

### מבנה קבצים
```
/
├── index.html          # עמוד ראשי
├── manifest.json       # Web App Manifest
├── sw.js              # Service Worker
├── logo.png           # לוגו האפליקציה
├── css/
│   └── style.css      # עיצוב מלא
└── js/
    ├── logic.js       # לוגיקת המשחק
    ├── ui.js          # ממשק משתמש
    ├── pwa.js         # ניהול PWA
    └── main.js        # נקודת כניסה
```

## 🎯 איך לשחק

### מטרת המשחק
להעביר את כל הקלפים לארבע ערימות הבסיס (Foundations), ממוינים לפי סדר עולה (Ace עד King) ולפי סימן.

### חוקים
1. **Tableau (7 עמודות)**: 
   - בנה מטה בסדר יורד וצבעים מתחלפים
   - רק מלך ניתן להניח בעמודה ריקה

2. **Foundations (4 ערימות)**:
   - התחל עם Ace
   - בנה למעלה באותו סימן (A→2→3...→K)

3. **Stock & Waste**:
   - שלוף קלפים מהקופה
   - העבר קלפים מה-Waste לטבלו או לבסיס

### טיפים
- נסה לפנות עמודות כדי לשים מלכים
- העדף העברה לבסיס כשאפשרי
- השתמש ב-Undo אם טעית
- השתמש ברמז אם תקוע

## 🐛 פתרון בעיות

### האפליקציה לא נטענת
1. בדוק חיבור אינטרנט (לטעינה ראשונית)
2. נסה לרענן את הדף
3. נקה Cache של הדפדפן
4. השתמש באיפוס קשיח

### אין שמירה אוטומטית
1. בדוק שהדפדפן מאפשר LocalStorage
2. וודא שיש מקום פנוי
3. נסה במצב גלישה רגיל (לא פרטי)

### אנימציות לא חלקות
1. כבה אנימציות בהגדרות
2. סגור טאבים אחרים
3. עדכן את הדפדפן
